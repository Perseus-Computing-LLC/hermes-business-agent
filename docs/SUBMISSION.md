# Submission Details for Nous Research Accelerated Business Hackathon

## Project Name
hermes-business-agent

## Elevator Pitch (under 200 characters)
hermes-business-agent empowers autonomous agents to execute real business operations, using NVIDIA for intelligent processing and Stripe for financial transactions.

## What it does
Our project is an autonomous agent built on the Hermes Agent framework. It's designed to perform complex business tasks that require both intelligent decision-making and the ability to interact with financial systems. For example, the agent can analyze market data to identify an opportunity, provision necessary cloud resources using its Stripe account, and then deploy a service to capitalize on that opportunity, all without human intervention.

## How we built it
- **Core Framework:** Hermes Agent
- **Intelligence:** NVIDIA NemoClaw for safe and powerful agent execution.
- **Financials:** Stripe Skills for Hermes to enable the agent to manage its own funds, pay for services, and generate revenue.
- **Language:** Python
- **Infrastructure:** Deployed on public cloud infrastructure.

## Why this hackathon
This hackathon is the perfect intersection of our team's expertise in building autonomous agents and the theme of creating real-world business value. The combination of Hermes Agent, NVIDIA's AI stack, and Stripe's financial infrastructure provides the exact toolkit needed to build agents that are not just intelligent, but also economically independent and capable of operating in the real world.
