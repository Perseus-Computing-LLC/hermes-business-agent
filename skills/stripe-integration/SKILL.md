# Stripe Integration Skill

This skill provides functions for interacting with the Stripe API, enabling Hermes agents to perform financial transactions.

## Features

- Create payment links
- Check payment status
- Manage products
- Manage subscriptions

## Setup

1.  **Stripe Account:** A Stripe account is required.
2.  **API Key:** The Stripe API key must be available as an environment variable (`STRIPE_API_KEY`).
